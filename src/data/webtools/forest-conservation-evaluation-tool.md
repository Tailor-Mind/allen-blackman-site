The Forest Conservation Evaluation Tool (FCET) helps users measure how effective existing forest conservation policies have been at preventing deforestation. It uses Global Forest Watch satellite imagery and statistical matching techniques to isolate the causal impact of conservation policies from site-selection effects.

Like the FCTT, the FCET is designed to be non-technical and easy to use, with embedded data and step-by-step instructions requiring no external downloads.
